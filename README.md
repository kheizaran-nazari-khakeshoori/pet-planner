# Pet Planner

A simple Python pet task manager with a Flask web UI and SQLite persistence.

## Run the app

```bash
cd /home/kheizaran/Github/pet-planner
source .venv/bin/activate
python app.py
```

Then open `http://127.0.0.1:5000`.

## Notes

- Pets and tasks are stored in `pet_planner.db`.
- The Flask UI currently shows pets and tasks.
